Improve and refine the website color system to look more elegant, modern, and professional.

==============================
COLOR REFINEMENT GOALS
==============================

- Keep the logo colors as branding reference
- Reduce color saturation for a softer, premium look
- Improve visual hierarchy and readability
- Avoid large solid color blocks

==============================
UPDATED COLOR STRATEGY
==============================

Primary Color (Green – refined):
- Use a softer dark green for navbar and headers
- Use lighter green shades for backgrounds and sections
- Avoid full-page green backgrounds

Secondary Color (Blue):
- Use blue for primary action buttons (Browse books, View details)
- Use blue for links and active states

Accent Color (Red):
- Use red ONLY for:
  - Prices
  - Important alerts
  - Order status badges
- Do NOT use red for large sections

Neutral Colors:
- White and off-white as main backgrounds
- Light gray for cards and separators
- Dark gray for text instead of pure black

==============================
SECTION IMPROVEMENTS
==============================

HOME HERO SECTION:
- Replace solid green background with:
  - Light gradient (green → white)
  - Or white background with green accents

FEATURE CARDS:
- White cards
- Soft shadow
- Rounded corners
- Colored icons only (not full background)

BOOK SECTIONS:
- White background
- Clear spacing
- Colored badges for categories

BUTTONS:
- Primary buttons: blue
- Secondary buttons: outlined green
- Hover states with smooth transitions

==============================
ADMIN DASHBOARD
==============================

- Use neutral light theme
- Brand colors only as accents
- Prioritize readability and clarity

==============================
TECHNICAL
==============================

- Implement colors using Tailwind theme configuration
- Use CSS variables for easy future changes
- Ensure Arabic RTL typography looks clean
- Maintain WCAG contrast standards

==============================
FINAL RESULT
==============================

Transform the UI into a clean, elegant, professional publishing house website with balanced colors and improved user experience.
